Minitroopers controller
==============================

Features:

* Crawl through all given trooper names
* Unlock missions
* Fight 3 times
* Go to Mission 3 times
* Raid 5 times
* Check if trooper needs upgrade
* In the end, return list of troopers that can upgrade

How to use
==========

* Clone repo
* `bundle`
* `ruby update.rb`
* Enjoy!
